# 📄 Resume Strength Analyzer

A modern and interactive web application that evaluates the overall strength of a resume based on essential career factors such as education, technical skills, work experience, projects, and certifications. The application calculates a resume score using a predefined weightage system and provides personalized feedback to help users understand their resume quality.

## 🚀 Features

- 🎓 Select Education Level
- 💻 Enter Number of Technical Skills
- 👨‍💼 Add Years of Work Experience
- 📂 Include Completed Projects
- 🏆 Enter Certifications Earned
- 📊 Instant Resume Score Calculation
- 📈 Resume Quality Evaluation
- ✨ Modern Glassmorphism User Interface
- 📱 Responsive Design
- ⚡ Real-time Results with Interactive Feedback

## 📊 Scoring Methodology

The application evaluates a resume using the following formula:

```text
Resume Score =
Education Score +
(Technical Skills × 2) +
(Work Experience × 3) +
(Projects × 2) +
(Certifications × 2)
```

### Education Weightage

| Education Level | Score |
|-----------------|------:|
| High School | 10 |
| Undergraduate | 20 |
| Postgraduate | 30 |
| PhD | 40 |

---

## 📈 Resume Evaluation Criteria

| Score Range | Evaluation |
|-------------|------------|
| **100 and above** | 🌟 Outstanding Resume – You stand out! |
| **70 – 99** | 👍 Strong Resume – Good chances! |
| **40 – 69** | 👌 Fair Resume – Needs some polishing. |
| **Below 40** | 💡 Weak Resume – Improve certain areas. |

## 💻 User Inputs

The application accepts the following information:

- Education Level
- Technical Skills Count
- Years of Work Experience
- Number of Projects
- Certifications Earned

Based on these inputs, the application generates a resume score and categorizes the resume into one of four performance levels.
